# Python Diary

## Description
Python Diary is a simple Python application designed to help you maintain a personal diary. You can use it to log your daily thoughts, ideas, and experiences.

## Features
- Add new diary entries
- View past entries
- Simple and intuitive interface

## Requirements
Make sure you have the required dependencies installed. You can find them in the `requirements.txt` file.

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd python-diary
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application using the following command:
```bash
python diary.py
```
